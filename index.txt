        <p class="main_title">WELCOME TO YAML.ORG</p>

        <p>YAML<sup><small>(tm)</small></sup> (rhymes with "camel")
          is a straightforward machine parsable data serialization 
          format designed for human readability and interaction with
          scripting languages such as Perl and Python. YAML is 
          optimized for data serialization, configuration settings,
          log files, Internet messaging and filtering. 
          YAML<sup><small>(tm)</small></sup> is a balance of the 
          following design goals: </p>
        <ul>
          <li>YAML documents are very readable by humans. 
          <li>YAML interacts well with scripting languages. 
          <li>YAML uses host languages' native data structures. 
          <li>YAML has a consistent information model. 
          <li>YAML enables stream-based processing. 
          <li>YAML is expressive and extensible. 
          <li>YAML is easy to implement. </li>
        </ul>
           <p>Please join us on the <a 
           href="http://lists.sourceforge.net/lists/listinfo/yaml-core">
           mailing list</a> or via <a href="irc://libera.chat/%23yaml">
           irc at #yaml on libera.chat</a>.
        </p>
